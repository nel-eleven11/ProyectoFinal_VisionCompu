# Modelos

Carpeta para pesos entrenados o descargados.

Archivos esperados mas adelante:

- `fretboard_yolo.pt`: modelo YOLO entrenado para detectar mastil, diapason, cuerdas o trastes.
- `chord_classifier_yolo.pt`: modelo YOLO de clasificacion entrenado con `dduka/guitar-chords`.
- `runs/`: salidas de entrenamiento de Ultralytics, si se entrena el detector dentro del proyecto.

Los pesos grandes normalmente se excluyen de Git y se documenta de donde descargarlos o como entrenarlos.
